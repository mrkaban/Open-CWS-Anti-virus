Introduction

MicroStar Anti-Virus is a fast, compact Anti-Virus for Microsoft(R) Windows(R) Machines.
An easy to use binary installer program is provided.

Basic Features:
  1.  Automatic Virus Database update via the internet.
  2.  Real-Time protection against viruses and other threats.
  3.  Run multiple scans at the same time.
  4.  Scanning of programs loaded in computer memory.
  5.  Easy and safe file quarantine.
  6.  Simple GUI
  7.  Scans within archives and compressed files (also protects against archive bombs).
  8. Built-in bytecode interpreter allows the MSAV signature writers to create and
    distribute very complex detection routines and remotely enhance the scanner’s
    functionality.

Usage

MicroStar Anti-Virus can be launched by selecting any of the installed shortcuts.
By default, during installation, a program group named "MicroStar Anti-Virus' is created.
Selecting "MicroStar Anti-Virus" from this program group will start MicroStar Anti-Virus, as shown below:



Different types of scans:

    1* Quick Scan : Quick Scan checks commonly affected areas for threats. This type of scan is fast. Click on the 'Run' button currusponding to the 'Quick System Scan' to launch this scan
    2* Full OS Scan : Full OS Scan checks the whole system drive for threats. This type of scan normally takes more than a hour to complete. Click on the 'Run' button currusponding to the 'Full OS Scan' to launch this scan.
    3* Memory Scan : This type of scan checks the programs and modules currently loaded in to the system memory. Click on the 'Run' button currusponding to the 'Full OS Scan' to launch this scan.
    4* Custom Scan : Custom scan only checks the files and folders you select. Click on the 'Run' button currusponding to the 'Custom Scan' to open the Custom Scan Window.

Updater

    Updater is used to keep the virus database up-to date. To help protect your PC, these databases are automatically updated.(requires internet connection)
    You can also update virus databases on demand.

Quarantine

    The infected files found by MicroStar Anti-Virus is quarantined for your protection. These files are stored in a special safe file format.
    In the case of a false-positive, you can restore the quarantined file using Quarantine Module.
    To open Quarantine:
             Tools--> Open Quarantine.

Real-Time Protection
   
    MicroStar Anti-Virus has a on-access virus scanner which automatically detects viruses hidden in files.
    This module monitors fixed disks and scans any new drives that are plugged in to the system.
    This module is automatically started during the startup and you dont need to launch MicroStar Anti-Virus to enable this protection module.

Uninstallation

If you wish to uninstall MicroStar Anti-Virus, Go to Control Panel--> Add or remove programs. Then, double click on MicroStar Anti-Virus.
Click on 'Yes' button in the dialog box that appears when you double click on MicroSar Anti-Virus.
You must restart your computer to completely uninstall MicroStar Anti-Virus.
